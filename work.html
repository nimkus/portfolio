Meet App

A modern, serverless Progressive Web Application (PWA) built with React that allows users to search for upcoming events using the Google Calendar API. Developed with a Test-Driven Development (TDD) approach, this app offers offline functionality, interactive data visualizations, and seamless cross-platform performance.
